        - Pg-hstore
        - Sequelize
        - Formidable
        - Nodemon
        - Fs
        - Jsftp
        - Cors
        - Dotenv
        - Express-session
        - Cookie-parser
    - Database
        - Postgres
        - pgAdmin
    - Infrastructure
        - Heroku
